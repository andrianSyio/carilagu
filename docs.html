Dokumentasi API Sonus

Gunakan Sonus API untuk mengidentifikasi lagu dari potongan audio atau video secara terprogram.

Endpoint Utama

Semua permintaan untuk identifikasi lagu dikirim ke endpoint tunggal ini.

POST https://lagu-neon.vercel.app/

Request Body

Request harus dikirim sebagai multipart/form-data.

Key Tipe Deskripsi
file File Wajib. File media (MP3, WAV, MP4, dll.) yang akan diidentifikasi.

Response

API akan selalu mengembalikan objek JSON.

Respons Sukses (200 OK)

Jika lagu berhasil dikenali.

{
  "success": true,
  "title": "Numb",
  "artist": "Linkin Park",
  "album": "Meteora",
  "cover_art": "https://url.to/cover.jpg",
  "preview_url": "https://url.to/preview.mp3"
}

Respons Gagal (4xx/5xx)

Jika terjadi kesalahan.

{
  "success": false,
  "message": "Lagu tidak dapat diidentifikasi."
}
curl -X POST \

  -F "file=@/path/to/your/song.mp4"
import requests

api_url = "https://lagu-neon.vercel.app/"
file_path = "path/to/your/song.mp4"

with open(file_path, 'rb') as f:
    files = {'file': f}
    try:
        response = requests.post(api_url, files=files)
        response.raise_for_status()
        print("Berhasil!")
        print(response.json())
    except requests.exceptions.RequestException as e:
        print(f"Gagal: {e}")
        print(response.text)
// Menggunakan Node.js dengan node-fetch
// npm install node-fetch@2 form-data

const fetch = require('node-fetch');
const FormData = require('form-data');
const fs = require('fs');

const apiUrl = "https://lagu-neon.vercel.app/";
const filePath = "path/to/your/song.mp4";

const form = new FormData();
form.append('file', fs.createReadStream(filePath));

fetch(apiUrl, { method: 'POST', body: form })
    .then(res => res.json())
    .then(data => console.log("Berhasil!", data))
    .catch(err => console.error("Gagal:", err));
<?php

$apiUrl = 'https://lagu-neon.vercel.app/';
$filePath = 'path/to/your/song.mp4';

$fileData = new CURLFile($filePath);
$postData = ['file' => $fileData];

$ch = curl_init();
curl_setopt($ch, CURLOPT_URL, $apiUrl);
curl_setopt($ch, CURLOPT_POST, 1);
curl_setopt($ch, CURLOPT_POSTFIELDS, $postData);
curl_setopt($ch, CURLOPT_RETURNTRANSFER, true);

$response = curl_exec($ch);
$httpCode = curl_getinfo($ch, CURLINFO_HTTP_CODE);
curl_close($ch);

if ($httpCode == 200) {
    echo "Berhasil!\n";
    print_r(json_decode($response, true));
} else {
    echo "Gagal dengan status code: {$httpCode}\n";
    echo $response;
}
?>
