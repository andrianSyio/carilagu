Kekuatan di Ujung Jari Anda

Sonus dirancang untuk memberikan kemudahan dan kecepatan dalam mengenali musik favorit Anda.

  • Sangat Cepat

    Proses identifikasi hanya butuh beberapa detik berkat teknologi audio fingerprinting yang canggih.

    Unggah berbagai format file, mulai dari MP3, WAV, hingga video MP4. Kami akan mengekstrak audionya.

  • API untuk Developer

    Integrasikan teknologi Sonus ke aplikasi Anda dengan API publik kami yang simpel dan mudah digunakan.
